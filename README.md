# orbit
ORBIT: an Open source Rust-based implementation of a Data build Tool, inspired by DBT

## Installation

- Install Python
- Clone this repo
- Install Rust
- Install `cargo-make`
- Run `cargo make`
- Run the CLI with `./orbit [options]`
